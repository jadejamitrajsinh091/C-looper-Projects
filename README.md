### 1️⃣ Alphabet Series (Q1)

**Description:**
Prints alphabets starting from `'a'` and increments by 4 each time.

**Example Output:**

```
a,e,i,m,q,u,y
```

**Logic:**

* Start from `'a'`
* Increment character by 4 (`alphabet += 4`)
* Continue until it is less than `'z'`
* Print comma between characters

---

### 2️⃣ Digit Counter (Q2)

**Description:**
Counts the total number of digits in a given number.

**Example:**

```
Input: 752
Output: Total number of digits: 3
```

**Logic:**

* Take integer input
* Convert negative number to positive
* Divide number by 10 repeatedly
* Count how many times division happens
* If number is 0 → count = 1

---

### 3️⃣ First & Last Digit Addition (Q3)

**Description:**
Finds the sum of the first and last digit of a number.

**Example:**

```
Input: 752
Output: 9
```

**Logic:**

* Last digit → `num % 10`
* First digit → divide number until single digit remains
* Add both digits
